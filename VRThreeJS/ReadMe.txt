This is the folder for my virtual reality project using Three.js.
To open my project follow the link on a smartphone in landscape orientation: unn-w16014375.newnumyspace.co.uk/animation/project.html
works better if you put your smartphone in VR headset.
This project enables you to veiw a basic solar system with a moon sun and earth. You can control the animation by looking at the buttons underneath the earth.
looking at the earth moon and sun will display basic information about the body.
In the folders you can find the source code
